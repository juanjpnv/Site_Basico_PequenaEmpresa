# Introdução

Apenas um layout básico utilizando apenas HTML e CSS.

Pode ser adaptado para pequenas empresas ou para site pessoal.

Foi desenvolvido de forma responsiva, para se adaptar a diferentes tamanhos de tela

# Tecnologias
* HTML5
* CSS3
